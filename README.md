# skycloudd/somtoday

A tool to get your somtoday schedule

## Install

`config.json`

```json
{
  "username": "your username",
  "password": "your password",
  "school_uuid": "school uuid from https://servers.somtoday.nl/organisaties.json",
  "client_id": "D50E0C06-32D1-4B41-A137-A9A850C892C2"
}
```

## Usage

```sh
chmod +x main.py
```
